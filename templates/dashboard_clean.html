¿Cómo leerlo?
Vista consolidada: cada cliente muestra el % facturado respecto al total anual (todas las fechas de entrega). El color indica la urgencia de la próxima entrega pendiente.
Lógica híbrida: Color base por días restantes. Si vas muy adelantado (+20%) mejora, si vas muy atrasado (-20%) empeora.
Colores: ● Verde = ≥90 días O muy adelantado | ● Amarillo = 30-89 días O situación mixta | ● Rojo = <30 días O muy atrasado | ● Vencido · % Esperado: (120 - días restantes) / 120 × 100
